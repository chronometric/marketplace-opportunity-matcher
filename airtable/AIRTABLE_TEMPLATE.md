# Airtable base template

## Template link (replace with your base)

After you create the **Opportunity Hub** base in Airtable:

1. Open the base → **Share** → **Create a shared link** (or publish a read-only gallery link if you prefer).
2. Paste the URL below and commit so collaborators can duplicate the structure.

```
https://airtable.com/invite/l/REPLACE_WITH_YOUR_TEMPLATE_LINK
```

> **Note:** Airtable sharing URLs are account-specific. This repository cannot ship a live template until you add your own link.

## Duplicate for a new workspace

- Use **Copy base** from the template (if you shared it as a template) or duplicate from your own master base.

## API

- Generate a **Personal Access Token** (Airtable newer auth) or legacy API key per [Airtable API docs](https://airtable.com/developers/web/api/introduction).
- Use the Base ID from **Help → API documentation** inside the base.
