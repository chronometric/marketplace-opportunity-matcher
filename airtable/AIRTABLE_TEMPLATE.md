# Airtable base template

## Setup checklist

1. Build the base using [`SETUP.md`](SETUP.md).
2. Confirm formulas in [`formulas.md`](formulas.md) and the **Top Opportunities** view (Score **>80**, last **24h**).
3. **Share** the base so others can duplicate it (see below).
4. Paste the duplicate/share URL in **Template link** below and commit.

## Template link (replace with your base)

After **Opportunity Hub** is ready:

1. Open the base → **Share** (or **…** menu) → create a link that allows **Copy base** / workspace duplicate (permissions depend on your Airtable plan).
2. Replace the placeholder below with your URL.

```
https://airtable.com/invite/l/REPLACE_WITH_YOUR_TEMPLATE_LINK
```

> URLs are account-specific. Replace `REPLACE_WITH_YOUR_TEMPLATE_LINK` when the base is published.

## Duplicate for a new workspace

- **Copy base** from the shared link, or duplicate from your master base in the Airtable home screen.

## API (Make / Zapier)

- Create a **Personal Access Token** with access to this base: [Airtable API](https://airtable.com/developers/web/api/introduction).
- **Base ID:** **Help → API documentation** inside the base (starts with `app...`).
